# Dashboard de Indicadores no Power BI

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aplicar os fundamentos do Power BI na construção de um dashboard interativo, transformando dados em informações organizadas e de fácil interpretação. Foram utilizadas técnicas de modelagem, criação de indicadores e visualizações para apoiar a análise dos dados.

## 🎯 Objetivos

- Aplicar os conceitos fundamentais do Power BI;
- Construir um dashboard organizado e intuitivo;
- Criar indicadores para facilitar a análise dos dados;
- Desenvolver visualizações que auxiliem na tomada de decisão.

## 🛠️ Tecnologias utilizadas

- Power BI
- Power Query
- Modelagem de Dados

## 📊 Funcionalidades

- Dashboard interativo;
- Indicadores (KPIs);
- Gráficos para análise dos dados;
- Segmentações para filtragem das informações;
- Organização visual das métricas.

## 💡 Principais aprendizados

Durante o desenvolvimento deste projeto foram praticados os seguintes conceitos:

- Importação e transformação de dados;
- Criação de medidas e indicadores;
- Organização de dashboards;
- Modelagem básica de dados;
- Desenvolvimento de visualizações analíticas.

## 📈 Insights obtidos

Com o dashboard é possível:

- acompanhar os principais indicadores;
- identificar tendências nos dados;
- comparar resultados entre diferentes categorias;
- apoiar a tomada de decisão baseada em dados.
